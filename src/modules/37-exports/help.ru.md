# Экспорты и Audit Packs

## Что такое Export Pack?

**Export Pack** — это структурированный пакет данных для внешних получателей: аудиторов, банков, налоговых консультантов или членов семьи.

### Типы пакетов

| Тип | Назначение | Типичное содержимое |
|-----|------------|---------------------|
| **Audit Pack** | Для внешнего аудита | GL Journal, Net Worth, Positions, Documents |
| **Tax Advisor Pack** | Для налогового консультанта | Gains/Losses, Dividends, Tax Lots |
| **Bank KYC Pack** | Для банка (KYC/AML) | Net Worth, Source of Funds, Beneficial Owners |
| **Operations Pack** | Для операционного учета | Payments, Reconciliations, Exceptions |

## Шаблоны (Templates)

Шаблоны позволяют переиспользовать конфигурацию пакетов:
- Предустановленные секции и фильтры
- Стандартные форматы для разных получателей
- Системные шаблоны: "Annual Audit Pack", "Quarterly Net Worth"

## Запуск экспорта (Export Run)

При запуске экспорта система:
1. Собирает данные из указанных секций
2. Применяет фильтры и client-safe маскирование
3. Генерирует файлы CSV и PDF
4. Сохраняет файлы в Document Vault
5. Фиксирует lineage snapshot

### Форматы файлов (MVP)
- **CSV** — табличные данные с разделителем
- **PDF** — титульная страница, список секций, дисклеймеры

## Data Lineage

Lineage отслеживает происхождение данных:
- **Sources** — коллекции-источники
- **As-of** — дата актуальности данных
- **Connectors** — внешние интеграции
- **DQ Score** — оценка качества данных (модуль 27)

## Client-Safe режим

Для внешних получателей данные маскируются:
- Номера счетов: `****1234`
- Внутренние заметки скрыты
- Staff-only поля исключены
- Детализация ограничена

## Sharing

Шаринг пакетов через защищенные ссылки:
- Интеграция с Consents (модуль 34)
- Срок действия ссылки
- Лимит скачиваний
- IP-фильтрация (опционально)
- Уведомления о доступе

## Расписания

MVP: сохранение расписаний с ручным запуском.
- Частота: ежедневно, еженедельно, ежемесячно
- Авто-шаринг после генерации
- Email-уведомления

## RBAC

| Роль | Права |
|------|-------|
| Admin/CFO/Controller | Создание пакетов, запуск экспортов, публикация |
| Compliance | Audit packs, проверка lineage |
| Relationship Manager | Создание shares для advisors |
| Client/Advisor | Только просмотр разрешенных shares |
