# Executive Home Dashboard

## Что показывает
Home Dashboard — центральная точка управления Multi-Family Office. Здесь агрегируются ключевые метрики, очереди задач и оповещения со всех модулей системы.

## Для кого
- **Owner / Admin**: полный доступ ко всем секциям
- **CIO**: инвестиционные метрики, риск-алерты, IPS breaches
- **CFO**: биллинг, платежи, AR/AP
- **Operations**: задачи, интеграции, data quality
- **Compliance**: IPS, риски, audit trail
- **RM / Advisor**: клиентские данные, сообщения
- **Client (client-safe)**: только Net Worth, документы, сообщения

## Как пользоваться

### KPI полоса
Кликните на любую карточку чтобы перейти к детализации:
- **Единый капитал** → модуль Net Worth
- **Открытые задачи** → список задач с фильтром status=open
- **На согласовании** → очередь approvals
- **IPS нарушения** → breaches с severity

### Мои очереди
Переключайте вкладки для просмотра разных типов элементов. Клик по строке открывает детальную карточку справа.

### Быстрые действия
Создавайте задачи, запросы, документы прямо с Home без перехода в другие модули.

## Source-first
Каждая цифра имеет:
- **as-of дату** — когда данные были синхронизированы
- **ссылку на источник** — откуда получены данные

## Audit-ready
Любое действие на Home пишется в audit trail:
- Создание задач
- Создание запросов
- Просмотр карточек

Просмотреть audit log можно в виджете внизу страницы или в модуле Security.
