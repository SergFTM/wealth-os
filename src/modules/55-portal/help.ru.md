# Клиентский портал (Module 55)

## Назначение
Отдельный client-safe контур для владельца состояния и членов семьи. Портал предоставляет доступ только к разрешённым данным и публикациям.

## Разделы портала

### Главная
Приветственная страница с KPI-карточками: чистая стоимость, доходность YTD, новые документы, открытые запросы, новые сообщения. Баннер объявлений.

### Net Worth
Опубликованные снимки чистой стоимости. Drill-down по категориям активов (client-safe). AI объясняет причины изменений.

### Performance
Опубликованные отчёты о доходности. Упрощённые графики. AI объясняет результаты.

### Документы
Только документы с флагом sharedToPortal. Категории, теги, скачивание. Логирование просмотров и скачиваний.

### Пакеты
Пакеты, опубликованные для клиента. Детальный просмотр состава пакета. Логирование скачиваний.

### Владение (Ownership)
Снимки ownershipViews только для чтения. Упрощённый граф структуры владения. Дисклеймеры trust/legal.

### Благотворительность
Опубликованные impact-отчёты. Нарративы на RU/EN/UK. Вложения.

### Отношения (Relationships)
Опубликованные карточки relationship-менеджеров. Ближайшие встречи. Без внутренних заметок.

### Запросы
Создание запросов: документы, платежи, отчётность, налоги, трасты, другое. Привязка к кейсам/задачам back-office. Отслеживание статуса.

### Согласия (Consent)
Сводка активных согласий. Запрос на изменение согласия.

### Сообщения
Клиентские треды с MFO. Вложения только из опубликованных документов. AI-помощник для черновиков.

### Аудит
Что клиент просматривал и скачивал за последние 30 дней. Прозрачность.

## Дисклеймеры
- **AI**: Выводы информационные и требуют проверки человеком
- **Tax**: Не является налоговой консультацией
- **Trust**: Не является юридической консультацией

## Авторизация (MVP)
- Email + PIN-код
- Сессионный cookie
- В будущем — SSO

## Языки
RU (по умолчанию), EN, UK
