# Инвестиционные идеи и Research Hub

Модуль для управления инвестиционными идеями, watchlists и исследовательскими заметками.

---

## Что такое инвестиционная идея?

**Инвестиционная идея** — это структурированное описание потенциальной инвестиции, включающее:

- **Тезис** — основная гипотеза, почему инвестиция может быть успешной
- **Катализаторы** — события или факторы, которые могут реализовать тезис
- **Риски** — факторы, которые могут привести к убыткам
- **План входа** — условия и зоны для открытия позиции
- **Размер позиции** — рекомендуемая аллокация
- **План выхода** — условия закрытия позиции (цели и стопы)

### Жизненный цикл идеи

1. **Черновик** — идея в разработке
2. **Активная** — идея готова для мониторинга
3. **В комитете** — идея направлена на рассмотрение инвестиционного комитета
4. **Одобрена** — комитет одобрил идею для реализации
5. **Отклонена** — комитет отклонил идею
6. **Архив** — идея закрыта (реализована или устарела)

---

## Watchlist

**Watchlist** — это список активов для отслеживания. Типы watchlists:

- **Тикеры** — акции и другие биржевые инструменты
- **Фонды** — взаимные фонды, ETF, хедж-фонды
- **Сделки** — потенциальные инвестиционные сделки
- **Менеджеры** — управляющие активами для due diligence
- **Пользовательский** — произвольный список

### Возможности watchlist

- Добавление элементов с заметками
- Связь с инвестиционными идеями
- Настройка ценовых алертов (опционально)
- Отображение базовых метрик (цена, изменение)

---

## Исследовательские заметки

**Research Notes** — заметки из различных источников для поддержки инвестиционных идей:

- **Статья** — аналитические статьи и публикации
- **Документ** — загруженные PDF, презентации
- **Звонок** — заметки с конференц-колла или беседы
- **Встреча** — протокол встречи
- **Отчёт** — исследовательский отчёт
- **Скриншот** — сохранённое изображение

Заметки можно связывать с идеями и использовать при генерации мемо.

---

## Инвестиционные мемо

**Мемо** — структурированный документ для представления идеи комитету:

### Разделы мемо

1. **Резюме для руководства** — краткое изложение идеи
2. **Инвестиционный тезис** — детальное обоснование
3. **Катализаторы и сроки** — что и когда
4. **Анализ рисков** — потенциальные угрозы
5. **Соответствие IPS** — проверка ограничений
6. **План реализации** — вход, размер, выход
7. **Рекомендация** — итоговое предложение комитету

### Аудитория мемо

- **Внутренний** — только для сотрудников
- **Для клиента** — безопасная версия для клиентского отчёта

## Связь с инвестиционным комитетом

Модуль интегрирован с модулем **Комитет** (28):

1. Создание пункта повестки из идеи
2. Прикрепление мемо к повестке
3. Запрос голосования
4. Отслеживание решения

### Статусы в комитете

- **На рассмотрении** — идея включена в повестку
- **Голосование** — идёт процесс голосования
- **Одобрено** — решение положительное
- **Отклонено** — решение отрицательное
- **Отложено** — требуется дополнительная информация

---

## Связь с IPS и рисками

### IPS (Investment Policy Statement)

Идеи проверяются на соответствие ограничениям IPS:

- Допустимые классы активов
- Лимиты концентрации
- Географические ограничения
- ESG требования

### Risk Module

Интеграция с модулем управления рисками:

- Оценка рискового уровня идеи
- Связь с риск-алертами
- Мониторинг нарушений

---

## Отслеживание результатов (Outcomes)

После реализации идеи отслеживайте результаты:

### Типы результатов

- **Мониторинг** — идея отслеживается без открытия позиции
- **Реализована** — позиция открыта
- **Закрыта** — позиция закрыта, результат зафиксирован

### Метрики результатов

- Цена входа / выхода
- Реализованная доходность (%)
- Срок удержания
- Выводы и уроки

## AI помощник

Встроенный AI-ассистент помогает с анализом идей:

### Доступные функции

1. **Резюме идеи** — краткое изложение ключевых аспектов
2. **Сравнение идей** — сопоставление двух идей по параметрам
3. **Проект мемо** — черновик меморандума для комитета
4. **Рекомендации** — предложения по улучшению идеи

### Формат ответов AI

Каждый ответ включает:
- **Контент** — основной текст анализа
- **Источники** — на каких данных основан анализ
- **Уверенность** — оценка надёжности анализа (%)
- **Допущения** — какие предположения сделаны
- **Дисклеймер** — обязательное предупреждение

## Быстрые действия

- **Создать идею** — новая инвестиционная идея
- **Создать watchlist** — новый список отслеживания
- **Добавить заметку** — новая исследовательская заметка
- **Сгенерировать мемо** — AI-генерация меморандума
- **Demo данные** — загрузить демонстрационные данные

---

## Рекомендации по работе

1. **Структурируйте идеи** — заполняйте все поля для полноты анализа
2. **Документируйте источники** — связывайте идеи с заметками
3. **Оценивайте риски** — каждая идея должна иметь анализ рисков
4. **Используйте комитет** — важные решения проводите через комитет
5. **Отслеживайте результаты** — анализируйте outcomes для улучшения процесса

---

## Дисклеймер

> **Важно:** Инвестиционные идеи носят информационный характер.
> Не является индивидуальной инвестиционной рекомендацией.
>
> Перед принятием инвестиционных решений проконсультируйтесь
> с квалифицированным финансовым советником.
