# Календарь и встречи

Единый календарь для управления встречами с семьёй, советниками, банками, комитетами и governance.

## Возможности модуля

### Календарь событий
- Визуальный календарь на неделю и месяц
- Цветовая маркировка по категориям: семья, советники, банки, комитеты, governance
- Быстрое создание событий
- Напоминания и уведомления

### Встречи (Meetings)
- Создание встреч с повесткой дня
- Приглашение участников с ролями
- Ссылки на видеоконференции
- Связь с governance meetings и committee meetings
- Статусы: запланировано, проведено, отменено

### Повестка дня (Agenda)
- Структурированные пункты повестки
- Владельцы и ответственные
- Статусы обсуждения: запланировано, обсуждено, отложено
- Связь с решениями governance

### Заметки (Notes)
- Markdown-форматирование
- Черновик и публикация
- Client-safe версия для семьи
- Вложения из Document Vault
- AI-суммаризация

### Action Items
- Задачи из встреч
- Ответственные и сроки
- Автоизвлечение из заметок с AI
- Связь с workflow tasks

## Категории событий

| Категория | Описание |
|-----------|----------|
| **family** | Семейные встречи |
| **advisor** | Встречи с советниками |
| **bank** | Встречи с банками |
| **committee** | Заседания комитетов |
| **governance** | Governance meetings |
| **ops** | Операционные встречи |

## Client-Safe режим

При включённом client-safe режиме:
- Показываются только события с флагом `clientSafeVisible`
- Скрываются внутренние участники и теги
- Заметки видны только с флагом `clientSafePublished`

## AI-ассистент

### Доступные действия
1. **Черновик повестки** — генерация повестки на основе темы встречи
2. **Суммаризация заметок** — краткое резюме в bullet points
3. **Извлечение action items** — автоматический поиск задач в тексте

### Метаданные AI
- Уверенность (confidence score)
- Допущения (assumptions)
- Источники данных

## RBAC и доступ

| Роль | Права |
|------|-------|
| **Admin/Ops** | Полное управление календарём |
| **Relationship Manager** | Создание встреч, публикация заметок |
| **Client** | Просмотр client-safe событий |
| **External Advisor** | Просмотр событий по приглашению |

## Audit

Автоматическая запись в audit trail:
- Создание/изменение события
- Добавление пунктов повестки
- Добавление/публикация заметок
- Создание/завершение action items
- Изменение конфигурации интеграций
