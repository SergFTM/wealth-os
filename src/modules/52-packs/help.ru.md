# Модуль 52: Пакеты для консультантов

## Обзор

Модуль "Пакеты" (Advisor Packs) позволяет формировать и безопасно передавать пакеты документов внешним получателям: аудиторам, банкам, налоговым консультантам, юристам и инвестиционным комитетам.

## Ключевые возможности

### Создание пакетов
- **Wizard-процесс**: пошаговый мастер создания пакета
- **Шаблоны**: готовые наборы для типовых запросов
- **Быстрый пакет**: создание по шаблону в один клик
- **AI-ассистент**: автоматический подбор документов и генерация cover letter

### Содержимое пакета
- **Документы**: файлы из Document Vault
- **Отчёты**: экспорты из модуля Exports
- **Снимки**: Ownership Map snapshots
- **Cover Letter**: сопроводительное письмо
- **Чеклист**: список недостающих документов

### Безопасность
- **Client-safe режим**: исключение внутренних заметок
- **Уровни чувствительности**: low / medium / high
- **Водяные знаки**: маркировка на скачиваемых файлах
- **Ссылки с TTL**: автоматическое истечение доступа
- **Лимиты скачивания**: ограничение количества загрузок
- **Опциональный пароль**: дополнительная защита

### Согласования
- **RBAC**: роли определяют право на утверждение
- **Workflow**: запрос → рассмотрение → утверждение/отклонение
- **Уведомления**: автоматические напоминания

### Аудит
- **Полная история**: кто создал, кто утвердил, кто скачал
- **Логирование доступа**: каждый просмотр и скачивание
- **IP-адреса**: с маскировкой для privacy

| Статус | Описание |
|--------|----------|
| draft | Черновик, редактируется |
| pending_approval | На согласовании |
| approved | Утверждён, готов к публикации |
| shared | Опубликован, ссылка активна |
| closed | Закрыт, ссылки отозваны |

## Создание пакета

### Шаг 1: Получатель
Укажите тип получателя, название организации, контактное лицо и цель запроса.

### Шаг 2: Scope
Выберите охват (household / entity / portfolio), период и режим client-safe.

### Шаг 3: Содержимое
Система автоматически предложит документы. Добавьте или исключите элементы вручную.

### Шаг 4: Обзор
Проверьте состав пакета, настройки чувствительности и водяных знаков.

### Шаг 5: Согласование
Запросите утверждение у ответственного лица. После утверждения можно опубликовать ссылку.

## AI-ассистент

### Возможности
- Подбор документов по типу получателя и цели
- Генерация сопроводительного письма
- Выявление недостающих документов
- Проверка на полноту

### Ограничения
- Требуется проверка человеком
- Не заменяет профессиональную экспертизу
- Работает только на основе имеющихся данных

## Публичный просмотр

Получатель переходит по ссылке и видит:
- Сопроводительное письмо
- Список документов с возможностью скачивания
- Информацию о сроке действия ссылки
- Уведомление о водяных знаках

Каждое действие логируется для аудита.

## FAQ

**Q: Как отозвать доступ?**
A: Откройте Share → нажмите "Отозвать". Ссылка немедленно перестанет работать.

**Q: Можно ли продлить срок ссылки?**
A: Нет, создайте новую ссылку с нужным сроком.

**Q: Что если документ обновился?**
A: Пакет содержит снимок на момент создания. Для актуальной версии создайте новый пакет.

**Q: Как узнать, скачали ли пакет?**
A: Вкладка Downloads показывает все события доступа.
