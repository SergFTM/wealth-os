# AI Copilot — Справка

## Обзор

AI Copilot — единый AI контур Wealth OS, доступный глобально через header и контекстно внутри модулей.

**Дисклеймер**: AI выводы информационные и требуют проверки человеком.

## Возможности

### Copilot Chat
- Задавайте вопросы по данным платформы
- Получайте структурированные ответы с источниками
- Сохраняйте ответы как drafts для дальнейшего использования

### Narratives
Автоматические резюме изменений:
- Net Worth: изменения активов, причины роста/падения
- Performance: доходность портфелей, сравнение с benchmark
- Risk: сводка алертов, концентрации, нарушения
- Liquidity: прогноз cash flow, критичные обязательства

### Drafts
AI-генерируемые черновики:
- Сообщения клиентам
- Committee packs для заседаний
- Policy summaries для IPS
- Report sections для отчетных пакетов

### Triage
Автоматическая сортировка задач:
- SLA под угрозой
- Проблемы качества данных
- Нарушения IPS
- Просроченные fees

### Feedback Loop
- Оценивайте ответы (thumbs up/down)
- Оставляйте комментарии для улучшения
- Feedback учитывается для развития AI

## Sources (Источники)

Каждый AI ответ содержит:
- **Sources** — ссылки на данные платформы
- **Assumptions** — предположения при анализе
- **Confidence** — уровень уверенности (0-100%)

Если источников недостаточно, AI сообщит: "Недостаточно данных".

## Client-Safe Mode

При включении client-safe режима:
- Скрываются internal notes и staff-only поля
- Ответы безопасны для показа клиенту
- Применяются redaction rules из aiPolicies

## Guardrails

AI не выполняет действия напрямую:
- Предлагает шаги и кнопку confirm
- Создает draft или task для подтверждения
- Все запросы проверяются RBAC

## Audit

Все AI ответы логируются:
- Prompt и response сохраняются
- Sources и confidence записываются
- Создается auditEvent для трассировки

## Типичные сценарии

### Объяснить рост net worth
1. Откройте Copilot
2. Нажмите "Explain net worth change"
3. Получите анализ с источниками

### Сводка risk alerts
1. Нажмите "Summarize risk"
2. AI покажет открытые алерты и приоритеты

### Draft письма клиенту
1. Нажмите "Draft client update"
2. Проверьте и отредактируйте draft
3. Отправьте через Comms

### Triage задач
1. Нажмите "Triage tasks"
2. Просмотрите предложения
3. Создайте tasks для выполнения

## RBAC

Доступ к AI зависит от роли:
- Owner/CIO: полный доступ
- CFO/Ops: финансовые и операционные запросы
- Compliance: compliance-related запросы
- Advisor: консультационные запросы
- Client: ограниченный client-safe режим

## Ограничения MVP

Текущая версия использует rules-based генератор:
- Структурированные ответы по шаблонам
- Источники из FileDB
- В будущем — интеграция с LLM
