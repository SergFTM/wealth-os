# Администрирование тенанта

## Зачем нужен Tenant Admin в MFO

Tenant Admin позволяет Multi-Family Office (MFO) и отдельным клиентам настраивать свою "версию" платформы Wealth OS:
- Собственный брендинг и визуальный стиль
- Настройки клиентского портала
- Локализация и терминология
- Политики и дисклеймеры
- Управление функциями

## Брендинг и White Label

### Цветовая палитра
- **Sage (Шалфей)** - основной зеленый цвет
- **Gold (Золото)** - акцентный цвет
- **Ivory (Слоновая кость)** - фоновый цвет
- **Gray (Серый)** - текст и границы

### Градиент
Выберите один из предустановленных градиентов:
- Шалфей → Золото (по умолчанию)
- Золото → Шалфей
- Слоновая кость
- Нейтральный

### Плотность интерфейса
- Комфортный - больше пространства
- Компактный - больше информации

## Портал клиента

Настройте, какие модули доступны клиентам:
- Сводка по капиталу
- Документы
- Коммуникации
- Отчеты
- Счета
- Задачи

## Языки и терминология

### Поддерживаемые языки
- Русский (по умолчанию)
- Английский
- Украинский

### Переопределения
Замените стандартные термины на специфичные для вашей организации.

## Уведомления и шаблоны

Настройте шаблоны email для:
- Готовность счета
- Просрочка счета
- Запрос документа
- Требуется одобрение
- Ревью доступа
- Отчет готов
- Назначение задачи
- Приветствие

## Policy Banners и дисклеймеры

Добавьте юридические предупреждения для модулей:
- Налоги - не является налоговой консультацией
- Трасты - требует юридической проверки
- AI - автоматические рекомендации
- API - MVP режим
- Отчеты - актуальность данных
- Глобальные - инвестиционные риски

## Feature Flags и Beta

Управляйте доступностью функций:
- AI Advanced features
- API Module
- Reporting Studio
- Risk Analytics
- Tax Harvesting
- Экспорт в PDF
- Массовые действия

## Домены и Dedicated URL

### Meta настройки (MVP)
- Выделенный URL (например, https://mycompany.wealthos.io)
- Кастомные домены с проверкой
- Регион AWS
- Режим изоляции

## Data Policies и Retention

- Срок хранения данных (по умолчанию 7 лет)
- Срок хранения аудит логов
- Регион хранения данных
- Режим приватности
- Настройки бэкапов
- Шифрование

---

⚠️ **Дисклеймер MVP**

White label настройки в MVP демонстрационные. Для production требуется:
- Настройка инфраструктуры
- Конфигурация DNS и SSL
- Настройка email сервера
- Согласование с командой DevOps
