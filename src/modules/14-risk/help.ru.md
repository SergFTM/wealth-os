# Риск и надзор

## Обзор

Модуль «Риск и надзор» предоставляет комплексный инструментарий для мониторинга и управления инвестиционными рисками портфелей. Включает анализ экспозиций, контроль концентраций, расчёт метрик риска, стресс-тестирование и систему оповещений.

> ⚠️ **Отказ от ответственности**: Риск метрики носят информационный характер и не являются инвестиционной рекомендацией.

## Основные функции

### Экспозиции (Exposures)

Анализ распределения портфеля по различным измерениям:

- **По классам активов** — акции, облигации, альтернативные инвестиции, денежные средства
- **По секторам** — технологии, здравоохранение, финансы, энергетика и др.
- **По географии** — США, Европа, Азия, развивающиеся рынки
- **По валютам** — USD, EUR, GBP, JPY и др.
- **По ликвидности** — высоколиквидные, среднеликвидные, низколиквидные

### Концентрации (Concentrations)

Мониторинг концентрации рисков:

- Концентрация по отдельным позициям
- Концентрация по эмитентам
- Концентрация по секторам
- Концентрация по странам
- Автоматическое сравнение с установленными лимитами

### Метрики риска (Risk Metrics)

Ключевые показатели риска:

| Метрика | Описание |
|---------|----------|
| **Volatility** | Стандартное отклонение доходности |
| **Max Drawdown** | Максимальная просадка от пика |
| **Beta** | Чувствительность к рынку |
| **VaR (95%)** | Value at Risk на уровне 95% |
| **CVaR** | Conditional VaR (Expected Shortfall) |
| **Sharpe Ratio** | Коэффициент Шарпа |
| **Sortino Ratio** | Коэффициент Сортино |
| **Tracking Error** | Ошибка отслеживания бенчмарка |

### Стресс-тесты (Stress Tests)

Сценарное моделирование:

- **Исторические сценарии** — 2008 кризис, COVID-2020, технологический крах 2022
- **Гипотетические сценарии** — рост ставок, инфляционный шок, геополитический кризис
- **Факторные шоки** — изменение курса валют, цен на сырьё, процентных ставок

### Алерты и оповещения (Alerts)

Система автоматических уведомлений:

- **Critical** — превышение лимитов VaR, крупные убытки
- **Warning** — приближение к лимитам, аномальная волатильность
- **Info** — плановые напоминания, обновления метрик

### Действия (Actions)

Отслеживание корректирующих мер:

- Ребалансировка портфеля
- Хеджирование рисков
- Эскалация на комитет
- Обновление лимитов

## Рабочие процессы

### Ежедневный мониторинг

1. Проверка дашборда KPI
2. Просмотр новых алертов
3. Анализ изменений экспозиций
4. Подтверждение/эскалация алертов

### Еженедельный обзор

1. Анализ метрик риска
2. Проверка концентраций
3. Запуск выборочных стресс-тестов
4. Обновление статуса действий

### Ежемесячный отчёт

1. Полный прогон стресс-тестов
2. Сводка по рисковым событиям
3. Анализ трендов метрик
4. Подготовка отчёта для комитета

## Интеграции

- **IPS модуль** — ограничения и лимиты из инвестиционной политики
- **Портфели** — данные о позициях и стоимости
- **Аудит** — все действия логируются в audit trail
